Deutsche Hilfe

TazPanel ist das SliTaz-Verwaltungswerkzeug. Damit kann das ganze System gesteuert werden wie: Pakete verwalten, Benutzer hinzufügen oder entfernen, "Live"-Systeme erzeugen und vieles mehr. Die verschiedenen Funktionen wählt man über die Werkzeugleiste am oberen Rand des Anwendungsfensters und in einigen Fällen über ein Untermenü oder eine Seitenleiste, wie z.B. bei der Paketverwaltung.

In den meisten Fällen gibt es in dem entsprechenden Abschnitt eine Beschreibung und nützliche Informationen. TazPanel verwendet neueste HTML 5 and CSS 3 -Technologie um eine übersichtliche und benutzerfreundliche Bedienoberfläche für alle Verwaltungsaufgaben in SliTaz anzubieten. Bei der Programmierung von TazPanel wurde auch von Anfang an der Schwerpunkt auf Verarbeitungsgeschwindigkeit und -qualität gelegt. Es unterstützt in vollem Umfang Internationalisierung.

TazPanel läuft als ein "web server", der standardmäßig an Port 82 Verbindungen akzeptiert und nur von dem lokalen System genutzt werden kann. Wenn das System von einem fernen Rechner administriert werden soll, kann die Konfigurationsdatei des "web server" so geändert werden, dass Verbindungen aus einem lokalen Netzwerk (LAN) oder aus dem ganzen Internet ("world wide web") zugelassen werden. Im letzten Fall sollte sichergestellt sein, dass für den Zugang ein starkes Kennwort gewählt wurde. Die Konfigurationsdatei des "web server" heißt /etc/slitaz/httpd.conf.

Terminal

TazPanel bietet eine einfache Terminal-Emulation, womit Kommandos mit root-Rechten auf dem fernen oder lokalen System ausgeführt werden können. Da nicht alle verfügbaren Kommandos benutzt werden können, sollte für die meisten Aufgaben eine SSH-Verbindung benutzt werden. Nichtsdestotrotz kann es nützlich sein, über die web-Oberfläche einige Kommandos ausführen zu können. Insbesondere wird wget unterstützt, damit Dateien von einem fernen Rechner transferiert werden können. Für den Transfer wird standardmäßig das Verzeichnis /var/www/downloads verwendet. In allen Kommandos können Optionen wie -R -l usw. verwendet werden.

Unterstützung anfordern

Unterstützung gibt es im SliTaz forum, bei der "mailing list" oder über den "IRC Gesprächskanal". Der offizielle SliTaz IRC Gesprächskanal befindet sich bei Freenode. Als "client" kann LostIRC, Xchat oder Pidgin unter SliTaz benutzt werden. Freenode bietet auch eine schöne IRC web-Oberfläche.

Beteiligen Sie sich!

Wenn Sie helfen und sich an der SliTaz-Entwicklung beteiligen wollen, könnte TazPanel das richtige für Sie sein! Wir brauchen Hilfe bei der Übersetzung, Gestaltung, Dokumentation und der Programmierung. Wenn Sie weitere Informationen benötigen oder einen neuer Graphik-Stil veröffentlichen wollen, können Sie die SliTaz Nutzergemeinschaft besuchen und an das Entwickler-Forum einsenden oder eine Korrektur mit einer Problemmeldung an einen an SliTaz Mitwirkenden schicken, so dass wir mit Ihnen in Verbindung treten können und Ihren Beitrag in TazPanel integrieren oder in ein separates Stil-Paket packen können.

Fehler und Funktionen

TazPanel ist noch eine junge Anwendung, die im Laufe der Zeit verbessert und aufpoliert werden wird. Wenn Sie einen Fehler finden, melden Sie ihn bitte bei: bugs.slitaz.org. Tazpanel hat einen Fehlerverfolgungsmodus, der in der Konfigurationsdatei tazpanel.conf aktiviert werden kann. Es bietet auch eine Fehlersuchseite.
